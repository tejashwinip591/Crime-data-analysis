🔌 API ENDPOINTS (6 TOTAL)
════════════════════════════════════════════════════════════════════════════════

1. GET /api/health
   Purpose: Server health check
   Returns: Status and timestamp

2. GET /api/crime/stats
   Purpose: Overall crime statistics
   Returns: Total crimes, crimes by type, solved cases, solve rate

3. GET /api/crime/trends
   Purpose: Crime trends analysis
   Returns: Trend data for 5 crime categories (2021-2024)

4. GET /api/crime/heatmap
   Purpose: Geographic crime visualization
   Returns: Heatmap data with coordinates and intensity

5. GET /api/crime/city/:city
   Purpose: Search crime data by city
   Returns: Agency information from FBI Crime Data API

6. GET /api/crime/agency/:agencyId
   Purpose: Search crime data by agency
   Returns: Agency-specific crime data

📞 SUPPORT & TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

Common Issues & Solutions:

Issue: npm command not found
Solution: Install Node.js from https://nodejs.org/

Issue: Port 5000 already in use
Solution: Change PORT in .env file or kill process

Issue: CORS errors in browser
Solution: Ensure backend is running on correct port

Issue: Charts not displaying
Solution: Refresh page, check console, verify internet

Issue: Search returns no results
Solution: Try major cities, check spelling

See SETUP.md for detailed troubleshooting section.
